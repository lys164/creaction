설계 노트 · 心理機制對照

每個 UI 元素對應的爽點機制(給團隊 sync 用)

① 資訊不對稱的逆轉 → 密碼鎖屏

密碼不是白給的,提示來自聊天中角色說漏嘴的細節。「破解成功」的瞬間動畫 = 猜疑兌換成確定性的儀式感。解鎖音效/震動是掌控感的具象化。

② 老虎機間歇性獎勵 → App 網格 + 平庸內容混排

每個 App 點開前只有紅點數字,不知道裡面有沒有料。列表裡刻意混入 60% 平庸日常(和媽媽的乾巴巴問候、繳費記錄),讓「挖到料」的那 40% 更炸。有料的條目不做任何視覺標記——必須自己翻。

③ 偵探感 → 線索筆記 + 跨 App 交叉驗證

單條記錄只是碎片:外賣「週二從不點晚餐」+ 小號「週二夜上傳便利店錄音」+ 搜尋記錄,三條線索拼起來才出真相。集滿 3 條線索解鎖「真相卡」,是「被我發現了」的正義+聰明雙重快感的變現點。

④ 曖昧不實錘 → 每個角色留一個未解決鉤子

游嶼通訊錄裡的「누나?」、沈先生兩年前另一把傘的照片、해수打了一半沒發出去的回覆。故意不給答案,寫進「下次記錄」的預告裡——焦慮回訪的引擎。

⑤ 進度可視化 → 關係階段條 + 下次記錄倒計時

每次查完,關係階段 +1、真相收集度上升、預告「2日後新記錄到達」。把打開動機從「怕錯過壞事」換成「故事在往前走」的追劇邏輯。

⑥ 韓國本地化 → 熟悉的 App 皮膚

聊天 = 카톡黃泡泡、外賣 = 배민薄荷綠、搜尋 = 네이버綠、記帳 = 토스藍、小號 = 부계정/뒷계文化、二手 = 당근。密碼 4 位數、생일/퇴근時間梗。UI 語言全韓文。

⑦ 付費結構 → 열람권(檢視券)

免費 App 建立日常質感和信任,鎖住的 1-2 個 App + 真相卡是付費點。券用完彈 paywall。每個角色的鎖點放在「情緒最高的位置」:游嶼鎖語音備忘錄(最甜)、沈先生鎖備忘錄+一張照片(最毛)、해수鎖真相(最反差)。
